# Flutter-kmstock-project

계명대학교 캡스톤디자인 주식초보자 도움 어플
<br>

>사용자에게 눈에 띄는 GUI를 제공하여 접근성을 높이고정보를 빠르게 제공하여<br>
장내 중요사항을 빨리 접할 수 있는 애플리케이션을 제공한다.<br>
이 본문에서는 어플리케이션에 대한 부분만 가지고 있다.<br>
Flutter version 2.5.2
<br>
<br>
